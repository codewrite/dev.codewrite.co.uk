Setting Up Jekyll

The first thing I did was installed Ubuntu on windows

Then I installed NodeJS and NPM (which I’m not sure if I needed).

And then installed Jekyll

To run it: jekyll serve --host=0.0.0.0

I edited .bashrc to add the GEM exports

1# Ruby exports
2export GEM_HOME=$HOME/gems
3export PATH=$HOME/gems/bin:$PATH

I added a couple of files to the _includes folder:

head.html

 1<head>
 2    <meta charset="utf-8">
 3    <meta http-equiv="X-UA-Compatible" content="IE=edge">
 4    <meta name="viewport" content="width=device-width, initial-scale=1">
 5    {%- seo -%}
 6    <link rel="stylesheet" href="{{ "/assets/main.css" | relative_url }}">
 7    <link rel="stylesheet" href="{{ "/assets/css/main.css" | relative_url }}">
 8    {%- feed_meta -%}
 9    {%- if jekyll.environment == 'production' and site.google_analytics -%}
10      {%- include google-analytics.html -%}
11    {%- endif -%}
12</head>

in the file above I added a link to my css file - which is built from main.scss in the assets/css folder.

and the in header.html I added if my_page.hidden != true so I could have posts thant wouldn’t get listed:

 1    <div class="trigger">
 2    {%- for path in page_paths -%}
 3        {%- assign my_page = site.pages | where: "path", path | first -%}
 4        {%- if my_page.title -%}
 5        {%- if my_page.hidden != true -%}
 6            <a class="page-link" href="{{ my_page.url | relative_url }}">{{ my_page.title | escape }}</a>
 7        {%- endif -%}
 8        {%- endif -%}
 9    {%- endfor -%}
10    </div>

I did a similar thing for home.html in the _layouts folder:

 1    <ul class="post-list">
 2      {%- assign date_format = site.minima.date_format | default: "%b %-d, %Y" -%}
 3      {%- for post in posts -%}
 4        {%- if post.hidden != true -%}
 5          <li>
 6            <span class="post-meta">{{ post.date | date: date_format }}</span>
 7            <h3>
 8              <a class="post-link" href="{{ post.url | relative_url }}">
 9                {{ post.title | escape }}
10              </a>
11            </h3>
12            {%- if site.show_excerpts -%}
13              {{ post.excerpt }}
14            {%- endif -%}
15          </li>
16        {%- endif -%}
17      {%- endfor -%}
18    </ul>

All these files came from the Github repo at the end of this article.

This is the base Jekyll theme. You can find out more info about customizing your Jekyll theme, as well as basic Jekyll usage documentation at jekyllrb.com

You can find the source code for Minima at GitHub: jekyll / minima

You can find the source code for Jekyll at GitHub: jekyll / jekyll
