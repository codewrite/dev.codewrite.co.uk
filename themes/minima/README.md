# Minima Theme - Hugo Conversion

This is a minimal Hugo theme inspired by Jekyll's Minima theme.

## Features

- Responsive design
- Clean, minimal styling
- Support for categories/tags
- RSS feed generation
- Google Analytics and Ads integration

## Layout Files

- `layouts/baseof.html` - Base template
- `layouts/index.html` - Home page
- `layouts/_default/single.html` - Single post/page
- `layouts/_default/page.html` - Page template
- `layouts/partials/header.html` - Header partial
- `layouts/partials/google-analytics.html` - Google Analytics
- `layouts/partials/google-ads.html` - Google Ads

## Static Assets

- `static/css/style.css` - Main stylesheet
